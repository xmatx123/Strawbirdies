# Web-Hitster

A multiplayer music timeline game where players compete to place songs in the correct chronological order.

## Features
- **Multiplayer Lobbies**: Create or join rooms to play with friends.
- **Timeline Gameplay**: guess where a song fits in your timeline of release years.
- **Spotify Integration**: (In Progress) Uses Spotify for song data and playback.

## Getting Started
1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Open the link displayed in the terminal.
